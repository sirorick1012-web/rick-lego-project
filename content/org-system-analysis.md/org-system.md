# The Analysis of Brickonize

  

 ## About Brickonize

Brickonize one of the sites that Lego collectors use to identify Lego parts, sets, and minifigs

[Brickonize](https://brickognize.com/)

  

 ### Breakdown

 We are going to look at four different topics about Brickonize.

  1. Resources being organized.

  2. Interactions the system supports.

  3. Classification system.

  4. Relationships between resources.

  

 #### Resources being organized

 The resources that are being utilized for this website are Lego bricks, parts, mini figs, Lego sets, and miscelanious parts.

 Almost every Lego part to every set can be searched on this site. This allows for people to check and see what parts go to which sets. So if you break a Lego set, and unsure of what pieces are missing, you can find them! This also applies to Lego minifigs. I feel that this site is more utilized to find out what the name of figs are, the year they were made, and parts if needed.

 This site utilizes another site called Bricklink. Bricklink is the asset database of everything Lego. So if you look a fig up,

 it will give you a link to Bricklink so you can see how much a fig is worth, the set it came in, the fig name, year it was made and buy parts if needed.

  

  ### Interactions with system supports.

  This site allows for you to either type in the name of the object being looked for or you can upload a picture, and best part, you can just simply snap a pic and get the answer instantly! This makes for a quick check of a fig before you buy it to make sure you are not overpaying, that the fig has the correct torso, head, legs, hair, and all the accessories. Very useful website for anyone that would like to start collecting or just see what their stuff is worth.

  

   ### Classification System

   This site utilizes several ways to classify Lego parts. If you snap a pic of a fig, it will show you year, name, and set. You can break it down futher by just scanning a torso. It will then show you a description of the torso and give the bricklink link. If you click on the link, you can see how many figs the torso goes to, the set it came from, pricke of just the torso, and price of the fig. I personally use this site to lookup Lego heads. I can see what all figs the head will go to and allows me to buy any missing parts. If the part you are looking up has several colors, you can look up by color.

   The way Brickonize allows searches, its almost impossible not to find what you are looking for.

  

   ### Relationships between Resources

   The core purpose of this site is to see what a piece/part is. If you want to know more, it uses hyperlinks to go to Bricklink. This site utilizes name and photos to identify what a part is. This site will also let you know if a part cannot be reconized, which means its either very rare or a possible fake, or a piece that Lego never made.

From a newbie to Lego to the hard core collector, Brickonize has made this a very user friendly searchable site with tons of information. I would def reccommend this site and it is a free site.
