# Collectible Mini Figures (CMF)

This CMF Series focuses on collectible figures in the LEGO world.
Today we are going to take a peak at what a cmf is. These are a special set of mini figs that are not found in a traditional lego set. These figures range from Marvel to DC to special figs that Lego feels like they should make. The sets come in whats called a "blind box", this means a sealed box that are a mystery. If you purchase a sealed case from Lego, there are three of each fig in the set in the case. If you buy retail, there is not a set number of figs in the case. Each set can contain 12 to 16 figs to complete the sets. Each series has what is called a "grail" fig. It will be the figure that is worth the most amount of money.
